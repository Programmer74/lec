\iframe{Псевдонимы команд (alias)}

В \textbf{некоторых} версиях sh, стандарте IEEE 1003.2 (POSIX.1), а также
современных интерпретаторах alias -- это возможность сделать короткую
текстовую замену: \vspace{0.5cm}
\begin{lstlisting}
$ alias x="/bin/ls -l "
$ alias y="/bin/ls -l"
$ alias z="/"
$ x z
=> /bin/ls /
$ y z
=> /bin/ls z
\end{lstlisting}
